Projects

iTask - Manage your todos at one place

iTask is a modern, feature-rich todo management application built with React, Tailwind CSS, and Framer Motion. It allows users to efficiently organize tasks with support for due dates, search, sorting, and progress tracking. The application includes advanced UX features such as animated navigation, undo delete functionality, a recycle bin for deleted tasks, and today/overdue task views. Tasks are persisted using localStorage, ensuring data remains intact across sessions. Designed with a clean, glowing UI and smooth micro-interactions, iTask focuses on both usability and visual polish.
